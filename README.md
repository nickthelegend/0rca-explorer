## Tech Stack

**Algorand** · **Next.js** · **React** · **Tailwind CSS** · **TypeScript**

## Getting Started

```bash
# clone
git clone https://github.com/nickthelegend/0rca-explorer.git
cd 0rca-explorer

pnpm install
pnpm run dev
pnpm run start
pnpm run build
```
